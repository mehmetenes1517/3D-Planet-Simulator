# Planet Simulator

------------------------------------------
## Installation

    git clone --recursive 'https://github.com/mehmetenes1517/3D-Planet-Simulator'
    cd 3D-Planet-Simulator/
    mkdir build/
    cd build/
    cmake ..
    make

You can start your game in <code>Bin/</code> folder after executing these commands

  - I Tried a new structure for OpenGL projects
  
  - Rendered 10k Asteroids spinning around jupyter object 
  
  - Optimized the rendering with GPU Instancing
